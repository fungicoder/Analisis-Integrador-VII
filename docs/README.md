# Dashboard de Análisis Integrador VII

He creado un dashboard interactivo para visualizar los datos del archivo CSV proporcionado.

## Cómo usarlo
1. Abre el archivo `index.html` en tu navegador web (Chrome, Edge, Firefox).
2. **No necesitas internet** para ver los datos, pero sí para cargar los iconos y la librería de procesamiento (PapaParse) la primera vez.

## Funcionalidades
- **Matriz de Sentimientos**: Visualiza rápidamente las respuestas positivas (verde), negativas (rojo) y neutrales (gris).
- **Análisis Automático**: El sistema busca palabras clave para determinar el tono.
- **Detalle**: Haz clic en cualquier celda para ver la respuesta completa.
